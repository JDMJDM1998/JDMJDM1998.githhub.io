My first Blog

第一步

访问Nodejs.org
下载Nodejs.pkg

第二步

打开terminal
开启管理员模式 sudo su
利用淘宝镜像源下载 cnpm 
    npm install -g cnpm --registry=https://registry.npm.taobao.org
利用cnpm 全局安装 hexo 
    cnpm install -g hexo

第三步

cd /usr/jdm/
创建Blog文件夹
    mkdir Blog
生成博客网站
    sudo hexo init
启动
    hexo s
